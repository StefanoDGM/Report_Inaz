COME USARE
1) Copia il file Excel del mese dentro la cartella `dist\input`.
2) Avvia `dist\ReportCommesse.exe`.
3) Il programma crea il file elaborato in `dist\output`.
4) Apri il file generato e filtra manualmente la colonna `% viaggio` in Excel.

STRUTTURA CARTELLE
- `dist\ReportCommesse.exe` = eseguibile da avviare
- `dist\input\` = cartella dove inserire il file sorgente del mese
- `dist\output\` = cartella dove trovare il file elaborato

FOGLI DEL FILE GENERATO
- `Stampa Commesse Dipendente`
  - copia completa del file sorgente
  - ore convertite in formato centesimale con 5 decimali

- `Riepilogo Viaggi`
  - raggruppa per dipendente e per giorno
  - calcola le ore lavorate totali
  - per le righe con `Descr.Reparto = MANUTENTORI` somma le ore di `COMMESSA` e `CHIUSURA`
  - per i `MANUTENTORI` il residuo pausa viene mostrato solo se il totale viaggio non copre 1 ora
  - per tutti gli altri reparti `Ore viaggio` e `Residuo netto` restano a `0`
  - la colonna `% viaggio` è pronta per il filtro manuale di Excel

NOTE
- Il file sorgente atteso deve essere un `.xlsx`.
- Se ci sono più file nella cartella `dist\input`, viene usato il più recente.
- Il file vecchio viene sostituito completamente dal nuovo output.
- L'eseguibile funziona come applicazione standalone su Windows.
